## Getting started for devs!
1. Fork or clone the repo and install node modules:
```bash
yarn install
```

2. Run the app:
```bash
yarn start
```

3. Open the browser and go to http://localhost:3000/

4. Happy coding!
Don't forget to star the repo and submit your pull requests!
